多节点实例¶

为了支持在机器上运行的每个应用程序实例的不同配置, 检查 NODE_APP_INSTANCE 环境变量, 并用于加载实例特定文件.

例如, 如果设置了 NODE_APP_INSTANCE=3 环境变量, 则在加载 default.json 之后立即加载一个名为 default-3.json 的文件.

这适用于所有配置文件, 因此如果您的 NODE_APP_INSTANCE=3 和 NODE_ENV=stage, 则在加载 stage.yaml 之后立即使用名为 stage-3.yaml 的文件.如果存在, 将加载.
