config-uncached¶

Sometimes for testing or DevOps, you need one script to load more than configuration. This is normally a challenge with node-config As a singleton, it returns the same configuration each time. To force node-config to reload the configuration, use the `config-uncached <https://www.npmjs.com/package/config-uncached>`_ module.
